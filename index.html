CSS list-style-type Property

The list-style-type property in CSS specifies the type of list item marker for ordered lists (ol), unordered lists (ul), and list items (li).

Unordered Lists (ul)

  • disc : A filled circle (default).
  • circle : A hollow circle.
  • square : A filled square.
  • none : No marker.

Ordered Lists (ol)

  • decimal : Numbers (1, 2, 3, ...).
  • decimal-leading-zero : Numbers with leading zeros (01, 02, 03, ...).
  • lower-alpha : Lowercase Latin letters (a, b, c, ...).
  • upper-alpha : Uppercase Latin letters (A, B, C, ...).
  • lower-latin : Lowercase Latin letters (a, b, c, ...).
  • upper-latin : Uppercase Latin letters (A, B, C, ...).
  • lower-greek : Lowercase Greek letters (α, β, γ, ...).
  • upper-greek : Uppercase Greek letters (Α, Β, Γ, ...).
  • lower-roman : Lowercase Roman numerals (i, ii, iii, ...).
  • upper-roman : Uppercase Roman numerals (I, II, III, ...).
  • armenian : Traditional Armenian numbering.
  • georgian : Traditional Georgian numbering (an, ban, gan, ...).
  • hebrew : Traditional Hebrew numbering.
  • cjk-ideographic : Plain ideographic numbers (一, 二, 三, ...).
  • hiragana : Hiragana syllables (あ, い, う, ...).
  • katakana : Katakana syllables (ア, イ, ウ, ...).
  • hiragana-iroha : Hiragana syllables in the Iroha ordering (い, ろ, は, ...).
  • katakana-iroha : Katakana syllables in the Iroha ordering (イ, ロ, ハ, ...).

Other Values

  • none : No marker.
  • inherit : Inherits the list style type from its parent element.
  • initial : Sets this property to its default value.
  • revert : Resets the property to its inherited value if it is inherited, or to its initial value if not.
  • unset : Resets the property to its inherited value if it is inherited, or to its initial value if not.

Extended Values (CSS3 and later)

  • lower-alpha : Alias for lower-latin.
  • upper-alpha : Alias for upper-latin.
  • disclosure-open : An open disclosure triangle, often used in trees and menus.
  • disclosure-closed : A closed disclosure triangle, often used in trees and menus.
